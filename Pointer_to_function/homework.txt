	ДОМАШНЕЕ ЗАДАНИЕ:
		1. Создайте функцию, принимающую в качестве параметров массив и
		указатель на функцию.
		2. Обеспечьте вызов функции-параметра в отдельности для каждого
		элемента массива.
		3. Функция-параметр должна возвращать число, основанное
		на текущем элементе массива (например, результат умножения его на 2)
		4. Основная функция должна вернуть результат сложения значений
		которые возвращаются из вызовов функции-параметра.
