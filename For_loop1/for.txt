Задание 1:
		1. Пользователь вводит числа
		2. Нужно посчитать количество введенных положительных и
		отрицательных чисел
		3. При вводе 0 программа завершается и показываем статистику
		ввода данных

		Задание 2:
		1. Программа "загадывает" число
		2. Пользователь вводит число
		3. Если введенное пользователем число соответствует числу
		которое "загадал" компьютер, тогда программа завершается,
		иначе число вводится заново
		4. Реализуйте режимы игры (Easy, Normal)
			- В случае Easy пользователю нужно выдавать подсказки(меньше/больше)
			- В случае Normal подсказки отсутствуют

			 123, 456, 789

		Задание 3:
		1. Программа "загадывает" три числа
		2. Пользователь должен "отгадать" их последовательно.
		Сначала 1-ое число, потом 2-ое число, и 3-е число
		3. Если пользователь допустит ошибку на одном из этапов ввода,
		тогда ввод чисел начинается заново (с первого числа) и так до тех пор
		пока все три числа не будут введены корректно
	
		ДОМАШНЕЕ ЗАДАНИЕ:
		
		Задание 4:
		1. Пользователь вводит диапазон букв (например, a-z)
		в качестве диапазона можно использовать только маленькие или
		большие буквы.
		2. Далее пользователь имеет право вводить только буквы из указанного
		диапазона и только в правильной последовательности.
		Например для a-z, первый ввод может быть a, но не b. 
		Только после ввода a можно ввести b и так далее.
		3. Когда ввод достигнет конца диапазона (в данном случае z),
		программа завершается.
